¿Qué son las Propiedades?

En Tailwind CSS, las propiedades son clases de utilidad que puedes aplicar a los elementos HTML para modificar su apariencia y comportamiento. Estas propiedades permiten personalizar fácilmente la tipografía, los colores, el espaciado, y mucho más.

Tipografía

Las clases de tipografía en Tailwind permiten controlar el tamaño, peso, y alineación del texto. Aquí algunos ejemplos:

Texto Grande

Este es un texto de tamaño grande.

Este es un texto con peso semibold.

Este texto está alineado a la derecha.

                
<h3 class="text-2xl font-bold">Texto Grande</h3>
<p class="text-lg">Este es un texto de tamaño grande.</p>
<p class="font-semibold">Este es un texto con peso semibold.</p>
<p class="text-right">Este texto está alineado a la derecha.</p>

Colores

Tailwind CSS proporciona una amplia paleta de colores que puedes utilizar para personalizar el fondo, el texto, y otros elementos. Aquí tienes algunos ejemplos:

Fondo Rojo
Fondo Verde
Fondo Azul
Fondo Amarillo
                
<div class="bg-red-500 text-white p-4">Fondo Rojo</div>
<div class="bg-green-500 text-white p-4">Fondo Verde</div>
<div class="bg-blue-500 text-white p-4">Fondo Azul</div>
<div class="bg-yellow-500 text-white p-4">Fondo Amarillo</div>

Espaciado (Margin & Padding)

Las clases de espaciado permiten controlar los márgenes y rellenos de los elementos. Aquí tienes algunos ejemplos de su uso:

Este div tiene un relleno y margen aplicados.
Este div solo tiene relleno.
                
<div class="bg-purple-500 text-white p-6 m-4">Este div tiene un relleno y margen aplicados.</div>
<div class="bg-purple-500 text-white p-8">Este div solo tiene relleno.</div>

Bordes

Puedes agregar bordes a tus elementos utilizando las clases de Tailwind. Aquí tienes ejemplos de bordes sólidos y redondeados:

Borde Sólido Azul
Borde Verde Redondeado
                
<div class="border-2 border-blue-500 p-4">Borde Sólido Azul</div>
<div class="border-2 border-green-500 rounded-lg p-4">Borde Verde Redondeado</div>

Sombras

Las sombras se pueden aplicar a los elementos para crear un efecto de elevación. Aquí hay algunos ejemplos:

Sombra Grande
Sombra Mediana
Sombra Pequeña
                
<div class="bg-gray-300 p-4 shadow-lg">Sombra Grande</div>
<div class="bg-gray-300 p-4 shadow-md mt-4">Sombra Mediana</div>
<div class="bg-gray-300 p-4 shadow-sm mt-4">Sombra Pequeña</div>
